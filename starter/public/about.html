Project Overview

This website was created as part of a Node.js project to demonstrate how a basic HTTP server handles requests and serves different types of content.

The focus of this project is on understanding server-side routing and response handling, rather than complex front-end design.

Key Features

  • Serving static HTML pages through Node.js
  • Handling routes for multiple pages
  • Delivering a shared CSS stylesheet
  • Providing custom error pages
  • Returning JSON data from an API endpoint
